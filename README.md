# Chess24HighlightExtension

Highlight your moves in live match on chess24.com!

**Installation**
1. ``` git clone https://github.com/uros-5/chess24HighlightExtension.git``` or download in zip and extract later.
2. In one tab open this: ```about:debugging#/runtime/this-firefox```
3. click on Load Temporary Add-on...
4. Navigate to downloaded repo and choose extension.js

Enjoy!

This extension works for Mozilla Firefox.
